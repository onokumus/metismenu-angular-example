Usage

1. Edit app.module.ts file

          
            import { MetismenuAngularModule } from "@metismenu/angular";

            @NgModule({
              imports: [
                MetismenuAngularModule,
                ...
              ]
          })

2. Edit angular.json file

        "styles": [
          "./node_modules/metismenujs/dist/metismenujs.css",
          "src/styles.css"
        ]

3. Use metis-menu selector

        <ul metis-menu>
